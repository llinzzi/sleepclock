# Sleep Clock PRD

## 1. Concept & Vision

一款床头氛围睡眠时钟。平日沉默如夜，按键唤醒时，柔和的灰阶渐显时间，仿佛月光从屏幕中渗出。不仅是时钟，更是一件能与环境对话的夜间艺术品。

结合双路氛围灯、舵机结构、全频扬声器，这是一台能"动"、能"响"、能"发光"的多感官唤醒设备。

## 2. Design Language

### 视觉风格
- **暗黑美学**: 平时屏幕完全熄灭，仅有微弱的状态指示灯
- **月相灵感**: 唤醒时使用深灰到浅灰的渐变，模拟月光质感
- **极简主义**: 只显示必要信息——时间，无多余元素

### 颜色方案（I4 灰阶）
- **纯黑** `0x0` - 背景，完全熄灭
- **深灰** `0x5` - 次要信息
- **中灰** `0x8` - 时间数字阴影
- **浅灰** `0xB` - 时间数字主体
- **亮灰** `0xF` - 高亮/强调

### 字体
- 时间显示: 粗体大号数字，浅灰色
- 状态指示: 小号点阵字符，深灰色

### 动效哲学
- **缓入**: 按键触发后 800ms 渐显，避免刺眼
- **呼吸**: 显示期间时间数字有微弱亮度呼吸感 (±5% 亮度周期 4s)
- **缓出**: 无操作 10 秒后 1200ms 渐隐回全黑

## 3. Layout & Structure

### 主界面（唤醒状态）
```
┌─────────────────────────────────┐
│                                 │
│                                 │
│          23:45                  │  ← 大号浅灰时间
│                                 │
│            ·                    │  ← 底部小点指示运行状态
└─────────────────────────────────┘
```

### 待机状态
- 屏幕完全熄灭 (0x0)
- 主板小LED指示运行状态（微弱闪烁）

## 4. Features

### 核心功能
| 功能 | 描述 | 触发 |
|------|------|------|
| 唤醒显示 | 按键唤醒，显示时间 + 呼吸动效 | GPIO3 按键 |
| 渐隐待机 | 10秒无操作后渐隐至全黑 | 软件定时器 |
| 深度睡眠 | 30分钟无操作进入深度睡眠模式 | 软件定时器 |

### 扩展功能（后续迭代）
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 闹钟 | 设定时间，到时渐亮唤醒 | P1 |
| 环境光感应 | 根据室内亮度调节屏幕亮度 | P2 |
| 无线同步 | NTP 校准时间 / 手机APP设置 | P3 |

## 6. 硬件利用思考

### 6.1 SSD1322 灰阶的创意用法

| 效果 | 实现方式 | 视觉描述 |
|------|----------|----------|
| 月光渗出 | 8级灰度渐显 | 时间从黑暗中缓缓浮现，如月光穿透云层 |
| 深度感 | 数字使用2-3层灰度叠加 | 时间数字有轻微3D浮雕感 |
| 氛围底噪 | 全屏极淡噪点纹理 | 避免纯黑死屏，增加有机感 |
| 呼吸光晕 | 整屏亮度±5%正弦波动 | 模拟呼吸节奏的微弱光效 |
| 星辰散落 | 随机淡灰点点缀背景 | 唤醒时短暂显示，如繁星闪烁 |

### 6.2 双路氛围灯 (GPIO20/21 → FM116C)

两路灯光可以独立控制，实现丰富的光效：

| 场景 | 灯光表现 | 氛围描述 |
|------|----------|----------|
| **日出唤醒** | 暖光(路1)从0%→100%在30分钟内逐渐亮起 | 模拟真实日出，温和唤醒 |
| **日落助眠** | 冷光(路2)先亮，暖光(路1)逐渐替代 | 模拟日落，促进褪黑素分泌 |
| **星辰投影** | 两路交替闪烁，模拟星光 | 将天花板变成星空 |
| **呼吸同步** | 两路灯光随呼吸节奏明暗交替 | 引导用户呼吸放松 |
| **环境跟随** | 灯光随OLED显示亮度联动 | 屏幕亮则灯亮，屏幕暗则灯暗 |
| **双色温阅读** | 冷暖两路混合调节色温 | 床头阅读灯模式 |
| **微光守护** | 一路保持微弱常亮(5%) | 夜间指引，不完全熄灯 |

> **功耗注意**: FM116C 是 H 桥驱动，适合驱动 LED 灯带或小功率灯泡，每路建议不超过 500mA。

### 6.3 扬声器 (GPIO4/5/6 → NS4168 I2S)

I2S 音频功放可以输出丰富的声音：

| 场景 | 声音表现 | 体验描述 |
|------|----------|----------|
| **自然唤醒** | 鸟鸣、溪流、风声渐起 | 比刺耳闹铃更符合生理节律 |
| **渐进闹铃** | 音量从 0% 在 5 分钟内渐升至 80% | 温和唤醒，不惊吓 |
| **白噪音助眠** | 雨声、海浪声、粉红噪音 | 屏蔽环境噪音帮助入睡 |
| **整点报时** | 每小时一次轻柔报时音 | 不看手机也知道时间 |
| **呼吸引导** | 特定频率提示音配合灯光 | 引导冥想/呼吸练习 |
| **闹铃音乐** | 内置几首轻音乐作为闹钟 | 可爱的复古闹铃声 |

> **省电**: 深度睡眠时关闭功放电源 (GPIO2 NS_CTRL)。

### 6.4 舵机结构 (GPIO9)

GPIO9 复用为舵机控制，可以做以下物理交互：

| 玩法 | 结构描述 | 体验描述 |
|------|----------|----------|
| **日出窗帘** | 舵机拉绳逐渐拉开窗帘 | 物理方式让阳光唤醒，比灯光更真实 |
| **纸鹤风车** | 舵机带动小型纸鹤/风车旋转 | 轻柔的视觉提醒，如梦境飘动 |
| **反光镜云朵** | 舵机旋转带动天花板反光点移动 | 星光投影效果，照亮屋顶 |
| **指针时钟** | 舵机带动时针/分针旋转 | 物理与数字结合的复古美感 |
| **轻敲提醒** | 舵机带动小锤轻敲桌面 | 无声的触觉提醒 |
| **翻页显示** | 舵机翻动小型翻页结构 | 每分钟翻一页，极简主义 |

> **注意**: GPIO9 是 ESP32-C3 的 Boot Strapping 引脚，上电时需保持正确电平。运行时作为舵机 PWM 控制没问题。

### 6.5 按键 (GPIO3)

| 交互 | 动作 |
|------|------|
| 单击 | 唤醒屏幕 + 氛围灯 |
| 双击 | 快速切换氛围灯场景 |
| 长按 3秒 | 进入设置模式 |
| 长按 10秒 | 强制重启 |

## 7. 场景组合

### 唤醒场景示例

**周一清晨 7:00 唤醒：**
1. **06:30** - 暖光开始 30 分钟缓慢渐亮（日出模拟）
2. **07:00** - 舵机轻拉窗帘，物理唤醒
3. **07:00** - 闹铃渐响，自然声音
4. **07:05** - OLED 屏幕渐显时间
5. **07:10** - 用户按键确认起床，灯光保留

**夜间入睡：**
1. 用户单击按键
2. 氛围灯切换暖光（路1:100%, 路2:20%）
3. OLED 显示时间 + 星辰背景动效
4. 白噪音/雨声播放
5. 10 秒无操作，灯光 + 屏幕渐隐
6. 30 分钟后进入深度睡眠

## 8. 技术约束

### 现有硬件
- ESP32-C3-MINI-1 (Xtensa, 160MHz)
- SSD1322 256x64 OLED (I4 灰阶)
- NS4168 I2S 音频功放
- FM116C 双路 H 桥 LED 驱动
- 舵机 × 1 (GPIO9)

### 实现要点
- LVGL v9.4.0 L8 格式渲染，内部转换为 I4 输出
- 按键去抖 (GPIO3)
- 软件定时器管理显示超时
- 渐变效果使用 LVGL 动画接口
- GPIO2 (NS_CTRL) 控制音频功放开关
- GPIO20/21 (IN1/IN2) PWM 控制灯光
- GPIO9 PWM 控制舵机
- I2S 总线 (GPIO4/5/6) 驱动 NS4168 音频
